Задача 2: Семантическая разметка

Создайте страницу "О себе" с использованием семантических тегов:

  1. header с вашим именем
  2. main с 2-3 абзацами о ваших интересах
  3. aside с контактной информацией
  4. footer с копирайтом
<header>Иван Иванов</header>
<main>
  <p>Я увлекаюсь программированием...</p>
</main>
<aside>Email: example@mail.com</aside>
<footer>© 2025</footer>

Задача 3: Списки и изображения

Создайте страницу с:

  • Упорядоченным списком из 3 любимых фильмов/книг
  • Неупорядоченным списком из 3 хобби
  • Изображением (можно использовать заглушку)
Пример:
  1. Фильм 1
  2. Фильм 2
  • Хобби 1
  • Хобби 2

Задача 4: Форма обратной связи

Создайте форму с полями:

Имя (текстовое поле)
Обязательное для заполнения
Email (поле для email)
С placeholder "example@mail.com"
Сообщение (textarea)
Минимум 10 символов
Кнопка отправки
<form>
  <input type="text" name="name" required>
  <input type="email" placeholder="example@mail.com">
  <textarea minlength="10"></textarea>
  <button type="submit">Отправить</button>
</form>

Задача 5*: Сложная структура

Создайте страницу блога с:

  • Шапкой (header) с навигацией (nav) и 3 ссылками
  • Основным содержанием (main) с 2 статьями (article), каждая с:
    • Заголовком (h2)
    • Датой публикации (time)
    • Изображением
    • Абзацем текста
  • Боковой панелью (aside) с поиском (form) и архивом (ul)
  • Подвалом (footer) с копирайтом

Выполните задания и сохраните файлы как:

  • task1.html
  • task2.html
  • ...

Для задач используйте любые тексты и изображения (можно-заглушки).
